# Meal plan

### How to run 
##### For Mac and Linux users 

    $ pip install virtualenv
    $ python3 -m venv venv
    $ . venv/bin/activate
    $ pip install --upgrade pip
    $ pip install -r requirements.txt
    $ export FLASK_APP=app
    $ flask run

Running on http://127.0.0.1:5000/


##### For windows users

    $ pip install -r requirements.txt
    $ set FLASK_APP=app
    $ flask run

Running on http://127.0.0.1:5000/
